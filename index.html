Explicando o Código

O "Doctype HTML" = é uma declaração para informar ao navegador qual é a versão do HTML utilizada no arquivo. Essa declaração vem antes das tags HTML, portanto, geralmente é apresentada na primeira linha de um código

"html lang="pt" = Sempre use um atributo de idioma no elemento "html". Este será herdado por todos os demais elementos, desse modo definindo o idioma padrão do texto no elemento head do documento.

"Head"= Em HTML, a tag é usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site. A tag head fica aninhada à tag html.e possui tag de abertura head e de fechamento /head, e tudo que estiver entre a abertura e o fechamento será considerado como fazendo parte do cabeçalho do documento.

"Title"A tag "Title" consiste na informação que vai aparecer na barra de navegação do seu navegador quando você entrar no site.

"Meta" = Meta tags são linhas de código HTML ou "etiquetas" que, entre outras coisas, descrevem o conteúdo do seu site para os buscadores. É nelas que você vai inserir as palavras-chave que facilitarão a vida do usuário na hora de te encontrar

"Meta charset" = O atributo charset serve para indicar o formato de codificação de caracteres utilizado no documento. ASCII, UTF-8, ANSI e ISO-8859-1 são exemplos de charsets. O charset UTF-8 é o que usamos na web atual e faz parte de um padrão chamado Unicode. A ideia é que com ele possamos representar qualquer caractere de qualquer idioma.

"Meta name" = Meta tags com o atributo name são utilizadas quando os tipos não correspondem com o HTTP, ou seja, aquilo que não se adequar ao HTTP-EQUIV é utilizado no name.

"Link rel=" = A tag link é uma tag bem generica como a tag meta. O atributo rel é utilizado para falar qual é a categoria do arquivo que estamos linkando, por isso passamos o valor stylesheet para o atributo rel quando estamos linkando uma forma de estilo no atributo href.

"script scr=" Script É usado para incluir ou referenciar um script executável. O "Scr" especifica o URI de um script externo; isso pode ser usado como uma alternativa para incorporar um script diretamente em um documento. Se um elemento de script tiver um atributo src especificado, ele não deverá ter um script incorporado em suas tags.

"Div class" = O elemento div é frequentemente usado como um contêiner para outros elementos, o que facilita na estilização de blocos. Div comporta receber os atributos ID e Class. Ou seja, é possível atribuir à uma div um ID único ou uma ou mais classes. Isso nos permite formatar um ou mais elementos de forma mais especifica e facilitada. Um ID é uma configuração única, ou seja, só podemos utilizar um ID em apenas um único elemento. Já uma classe é uma configuração que pode ser utilizada em mais de um elemento.
